Usage:
put the gamestate_integration_Dota_Cue.cfg in your Dota2 config folder (ex D:\Steam\steamapps\common\dota 2 beta\dota\cfg)
-h: list commands
-s: asks to enter ip (ex. 72.111.1.25:3000 or localhost:3000) then starts the game listener.
-quit: quits the application


Using:
Dota2GSI: https://github.com/antonpup/Dota2GSI
CUE.NET: https://github.com/DarthAffe/CUE.NET
